Gap Filling in Images - Modified Direct Sampling Approach / Julia Code

Project Description

The present work is considered an optimized version of the gap filling algorithm that was developed in my M.Sc. thesis.
In this version, I have re-written the code in Julia instead of Python. Moreover, I have expanded the scope of the study by testing more bands from Sentinel 2 imagery (for a total of 10) and testing six bands from Landsat 7 (before and after the instrument failure).
